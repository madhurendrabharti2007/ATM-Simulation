# ATM-Simulation system
A Python-based ATM simulation system with multi-user support, PIN generation, OTP verification, secure login (3 attempts), and transaction history using file handling and JSON.

## Features
- Multiple users (username + PIN)
- PIN generation
- OTP verification
- 3 login attempts security
- Deposit, Withdraw, Balance check
- Transaction history

## Technologies Used
- Python
- File Handling
- JSON

## How to Run
1. Run main.py
2. Register user
3. Login and use ATM features
